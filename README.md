## Usage

### Dependencies

1. [OpenCV](https://opencv.org/) –– open source CV library
2. [plog](./plog) –– lightweight logging library

### Git submodules initialization
```
git submodule update --init --recursive
```

### Building
To build source code use cmake:
```
mkdir build
cd build
cmake ..
make 
```

### Running
```
cd build
./main
```
